      Como Jogar

      Preencha as letras nas células para formar as palavras com base nas pistas.
      As letras são inseridas da esquerda para a direita em cada palavra.
      Use as setas do teclado ou clique nas células para navegar.
      Se pressionar Backspace em uma célula vazia, o foco retorna para a célula anterior do mesmo bloco, permitindo deleção contínua.
      Seu progresso é salvo automaticamente!
